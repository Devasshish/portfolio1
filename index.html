Vehicle Reselling Platform

HTML
CSS
JavaScript
PHP
MySQL

I developed a Vehicle Reselling Website that connects buyers with verified showrooms to purchase pre-owned vehicles. The system focuses on ensuring trust, transparency, and ease of use, especially for users looking for quality-checked second-hand cars.

Admin & Dealer Dashboard: The platform includes a backend dashboard where admins and registered dealers can upload vehicle details, manage listings, and track user inquiries. It also includes basic analytics for performance tracking.

Buyer-Friendly Interface: The frontend is designed for easy navigation, allowing users to browse available cars, filter by brand, price, and condition, and view detailed information with images, specifications, and contact details.

Verified Listings by Showrooms: Every vehicle listed on the platform is first verified by trusted showroom partners to ensure authenticity, proper documentation, and mechanical condition before being published for sale.

Hospital Management System

VB.NET
SQL Server
Windows Forms

To simplify and streamline hospital operations, I developed a custom Hospital Management System using VB.NET. This system was designed to efficiently manage patient records, doctor schedules, billing, appointments, and reporting — all in one desktop application.

Centralized Patient Records: All patient details including medical history, appointments, and treatment logs are stored securely in a structured SQL Server database, ensuring quick access and organized record-keeping.

Automated Billing & Reports: The system automatically generates invoices based on treatments, services, and medication. It also provides downloadable reports for daily transactions, patient history, and appointments.

Doctor & Appointment Scheduling: A built-in module manages doctor availability and appointment bookings, reducing manual errors and improving overall workflow efficiency.
